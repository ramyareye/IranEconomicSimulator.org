# `chatgpt/` (ChatGPT playbooks and copy)

This folder contains **ChatGPT-authored** documentation: public-site copy guidance, prompt templates, and quality checklists.

It is meant to **accelerate writing and content production** without weakening the project’s credibility standards.

If anything here conflicts with `codex/`, **`codex/` wins**.

If you want the “super detailed, don’t miss anything” version with sample release data, start with `chatgpt-complete/README.md`.

## Files

- `chatgpt/FOLDER_REVIEW.md` — review notes on `docs/` vs `codex/` and recommended next steps
- `chatgpt/PUBLIC_SITE_COPY.md` — recommended public narrative, page outlines, and microcopy rules
- `chatgpt/CHATGPT_WORKFLOW.md` — how to use ChatGPT safely for data + docs work (no hallucinated sources)
- `chatgpt/PROMPT_PLAYBOOK.md` — reusable prompt templates (metrics/events/sources/translation)
- `chatgpt/QUALITY_CHECKLIST.md` — ship checklist for data, scenarios, and public copy
