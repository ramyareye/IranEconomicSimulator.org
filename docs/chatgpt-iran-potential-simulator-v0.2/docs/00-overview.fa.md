# مرور کلی پروژه (FA)

این داکیومنت توضیح می‌دهد IPS چیست، چرا «سناریو» است، و چطور از نسخه ۰.۱ شروع می‌کنیم.

## اصول
- هر عدد باید منبع یا روش تخمین داشته باشد
- هر پارامتر بازه عدم قطعیت دارد (min/max)
- هر خروجی باید حساسیت‌سنجی داشته باشد
- داده‌ها و مدل نسخه‌بندی می‌شوند

## خروجی‌های نسخه ۰.۱
- مسیر GDP در ۵/۱۰/۱۵ سال
- GDP سرانه
- درآمد دولت (Tax + Oil + Customs - Leakage)
- خروجی JSON برای مصرف وب/موبایل

## چرا Git-first؟
چون تو می‌خوای از توییتر/HN شروع کنی و اعتماد بسازی؛ Git یعنی:
- تاریخچه
- review
- مشارکت جامعه
- شفافیت
