# Iran Regime Economic Cost Tracker

**Live / Static Ticker of Cumulative Economic Waste Since 1979 + Counterfactual Future if Regime Ends**

**هدف پروژه**: نمایش شفاف هزینه‌های هدررفته، فساد، پروکسی‌ها، جنگ، تحریم‌های خودساخته و فرصت‌های از دست رفته از انقلاب ۵۷ تا امروز + سناریوهای رشد GDP در صورت تغییر رژیم و بازگشت به اقتصاد عادی.

**Current Estimate (March 2026)**: ~12–15 تریلیون دلار cumulative opportunity cost (بر اساس synthetic control studies + wiiw report 2026 + دیگر منابع).
