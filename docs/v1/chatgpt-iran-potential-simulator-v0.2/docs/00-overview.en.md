# Project overview (EN)

IPS is an open, assumption-driven economic scenario simulator. It is designed for transparency and reproducibility (versioned JSON, sources, ranges, sensitivity).

## v0.1 outputs
- GDP path (5/10/15 years)
- GDP per capita
- Government revenue (Tax + Oil + Customs - Leakage)
- JSON outputs for web/mobile

## Why Git-first?
To build trust early (Twitter/HN): reviewable history, community PRs, and clear provenance.
