# هزینه جمهوری اسلامی — Cost of the Islamic Republic
## پروژه آزادی ایران | Iran Freedom Dividend Project

> **یک تراکر زنده و مستند که نشان می‌دهد جمهوری اسلامی چقدر از ایران و ایرانیان هزینه گرفته است.**
>
> *A live, fully-sourced tracker showing the true cost the Islamic Republic has imposed on Iran and its people.*

---

## 🧭 Vision & Positioning

### برای چه کسی؟
- **اول:** هکرها، ژورنالیست‌ها، دیاسپورای ایرانی (Twitter/HN/Reddit)
- **دوم:** ایرانیان داخل و خارج از کشور که می‌خواهند آمار مستند داشته باشند
- **سوم:** پژوهشگران، روزنامه‌نگاران، سیاست‌مداران

### چه چیزی این پروژه را متفاوت می‌کند؟
برخلاف `iran-cost-ticker.com` که هزینه جنگ آمریکا را نشان می‌دهد، این پروژه نشان می‌دهد **ایرانیان** چه پرداخته‌اند — نه آمریکایی‌ها.

**The key insight from HN:** The original tracker faced criticism for vague methodology and missing interceptor costs. We will build defensible, transparent, source-linked numbers from day one. Every number has a source. Every estimate has an explicit methodology note.

## 🏗️ Technical Architecture

```
CDN:              Cloudflare
```

### Why GitHub JSON for data?

فوائد این رویکرد:
1. **شفافیت کامل** — همه می‌توانند داده‌ها را بررسی و fork کنند
2. **Pull Request as Data Review** — هر تغییر داده یک PR است با history
3. **Community contributions** — دیگران می‌توانند داده اضافه کنند
4. **Versioned methodology** — هر بار عدد عوض می‌شود، چرا واضح است
5. **No database needed initially** — می‌توان با فایل‌های JSON شروع کرد

**Recommended hybrid approach:**
- Static/research data: GitHub JSON (metrics, comparisons, events)
- Live data (exchange rates, oil price): Cache in Redis/DB, but export JSON snapshot daily
- All snapshots: Committed to GitHub nightly via GitHub Actions

## 🎯 Methodology Principles (Critical for HN credibility)

**درس اصلی از HN thread:** مردم بلافاصله روش‌شناسی را زیر سؤال می‌برند. پاسخ ما:

1. **Every number has a source link** — no orphan statistics
2. **Explicit confidence levels:** `high | medium | estimate | speculative`
3. **Range estimates** where point estimates are misleading: "IRGC budget: $20-35B"
4. **Counterfactual methodology documented** — how we pick peer countries and why
5. **Separate "direct costs" from "opportunity costs"** clearly labeled
6. **Welcome corrections** — GitHub Issues for data challenges
7. **Don't overstate** — conservative estimates with notes on why real numbers may be higher

**Confidence Tiers:**
- 🟢 High: World Bank / IMF / UN official data
- 🟡 Medium: Think tank estimates (RAND, Brookings, Atlantic Council)
- 🔴 Estimate: Our own calculations from primary data
- ⚫ Speculative: Best guess with explicit uncertainty range
