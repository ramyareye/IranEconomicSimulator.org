# مدل (FA) — نسخه ۰.۱

## تعریف سناریو
سناریو مجموعه‌ای از پارامترهاست که به مدل داده می‌شود.

### رشد GDP
GDP_t = GDP_{t-1} * (1 + g_t)

g_t از ماژول‌ها می‌آید:
- g_base: رشد پایه
- g_trade: اثر تجارت/اتصال مالی
- g_governance: اثر اصلاح حکمرانی/کاهش فساد
- g_competition: اثر شکستن انحصارها
- g_capital: بازگشت سرمایه/کاهش ریسک
- g_talent: بازگشت نیروی متخصص/بهره‌وری
- g_oil: اثر ظرفیت نفت/گاز
- shocks: شوک‌های احتمالی (اختیاری)

## درآمد دولت
Rev_t = Tax_t + OilRev_t + Customs_t + SOE_t - Leakage_t
Tax_t = tax_to_gdp_t * GDP_t
